## Recording Tips

1. **Device**: Use a clean emulator or physical device
2. **Orientation**: Portrait mode
3. **Resolution**: 1080p minimum
4. **Frame Rate**: 60fps for smooth animations
5. **Audio**: Clear voiceover or background music
6. **Duration**: Keep it under 1 minute
7. **Editing**: Add text overlays for key features

## Screen Recording Commands

### Android (ADB)
```bash
adb shell screenrecord /sdcard/demo.mp4
# Record for 60 seconds
adb pull /sdcard/demo.mp4
```

### iOS (QuickTime)
1. Connect device
2. Open QuickTime Player
3. File → New Movie Recording
4. Select device
5. Record

### Emulator
- Use built-in screen recording
- Or use OBS Studio for better quality

## Post-Production
